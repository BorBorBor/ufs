{% trans "What do the transaction status terms mean?" %}

{% trans "Registered" %}
{% blocktrans %}The transaction has been registered as a valid transaction with correct debit and credit entries, in this state no accounts have been affected.{% endblocktrans %}
{% trans "Payed" %}
{% blocktrans %}The debit side of the transaction has been confirmed and should now have been done in the real world.{% endblocktrans %}
{% trans "Received" %}
{% blocktrans %}The credit side of the transaction has registered that the money has been received in the real world.{% endblocktrans %}
